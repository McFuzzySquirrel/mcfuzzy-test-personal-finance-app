# Student Finance App

Phase 1 foundation scaffold for the student personal finance app.

## Stack

- Expo SDK `55.0.6`
- React Native + TypeScript (strict mode)
- React Navigation v7 (bottom tabs + native stack)
- SQLite via `expo-sqlite` v2

## Project Structure

```text
app/
components/
db/
hooks/
store/
utils/
types/
__tests__/
e2e/
```

## Setup

```bash
npm install
```

## Run

```bash
npm run start
npm run android
```

## Quality Checks

```bash
npm run lint
npm run typecheck
npm run test
```

## E2E

Detox is scaffolded as a placeholder command in this phase.

```bash
npm run e2e
```
